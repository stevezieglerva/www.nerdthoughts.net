But, I get bothered when I see a Christmas picture in July. Here are instructions on how to use Picasa and Dropbox to create a seasonal background slideshow synchronized across computers.


  • In Picasa, identify the pictures you want to tag by first starring them as a favorite. I usually usually flip through folders of new pictures and star them as a fast way to identify pictures I want to tag. 

  • Filter to show the starred pictures using the star filter at the top of the window.


  • Select the pictures using typical Windows CTRL or SHIFT clicking. 

  • Press CTRL-T to bring up the Tag pane. Type in "summerxtag"  and click the plus button to tag the photos for the summer season. I add "xtag" to the tag names to ensure the full tag word is unique across all tags and folder names in Picasa.  Picasa has some annoying search behavior if the tag is not unique.


  • If you have already tagged other pics for the summer, search on "summerxtag" in the filter bar. 

  • Select the "Search results for" album to show all of the matching pictures, no matter what folder they are in.



  • Click the Export button to export the pictures. Choose to export the pictures to a shared Dropbox folder. If you have really hi-res pictures, you can reduce the resolution to 1200 or 1600.


  • For each computer to synchronize, right mouse click on the Windows desktop and choose Personalize. Click Desktop Background.

  • Click the Browse button to choose your shared Dropbox folder with the picture exports. Be sure to select all pictures. 




The seasonal pictures will now rotate on the desktop. And, any future updates to the pictures will automatically be synchronized to all of the computers.
