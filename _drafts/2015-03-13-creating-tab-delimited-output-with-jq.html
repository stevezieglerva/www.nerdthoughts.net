--- layout: post title: Creating Tab Delimited Output With JQ In DOS date: '2015-03-13T10:00:00.000-04:00' author: Stephen Ziegler tags: modified_time: '2015-03-13T10:00:01.019-04:00' blogger_id: tag:blogger.com,1999:blog-1727955271917225446.post-5785301212627702207 blogger_orig_url: http://www.nerdthoughts.net/2015/03/creating-tab-delimited-output-with-jq.html --- I've been using jq to play around with JSON data returned from webservices. It's a great command line tool for manipulating and prettifying JSON.

This post describes how to get tab separated values out of jq on UNIX.  But, the format and escaping is different if you're trying it in DOS.  Here's an example to iterate through some Elasticsearch results (aggregations and buckets arrays) in the temp_results.json file to create a counts.txt file with the key and doc_count fields separated by a tab.



jq -r ".aggregations[] | .buckets[] | \"\(.key)\t\(.doc_count)\"" temp_results.json > counts.txt
